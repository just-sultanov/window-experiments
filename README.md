# window-rs

Experiments with window creation (Rust, C++ and Metal)

## Examples

### Metal (C++)

```bash 
# Go to the directory with examples
$ cd examples

# Compile the examples
$ make build

# Run the `hello world` example
$ ./build/hello-world
```
